# Chess Sound Assets

This directory contains sound effects for the chess application.

## Required Sound Files

Please add the following MP3 or WAV files to this directory:

1. **move.mp3** - Standard piece move sound (subtle "tap" or "click", ~0.3s)
2. **capture.mp3** - Piece capture sound (sharper "thud", ~0.4s)
3. **check.mp3** - Check alert sound (warning chime, ~0.5s)
4. **checkmate.mp3** - Checkmate/game end sound (victory/defeat tone, ~1s)
5. **castle.mp3** - Castling move sound (unique "whoosh" or double-tap, ~0.5s)
6. **promotion.mp3** - Pawn promotion sound (ascending chime, ~0.6s)
7. **illegal.mp3** - Illegal move attempt (error buzz, ~0.3s)
8. **game-start.mp3** - Game start/reset sound (opening chime, ~0.7s)
9. **button-click.mp3** - UI button click (subtle click, ~0.2s)

## Sound Characteristics

All sounds should be:
- Short duration (< 1 second except checkmate)
- Clean and professional quality
- Appropriate volume (not too loud)
- Royalty-free or licensed for use
